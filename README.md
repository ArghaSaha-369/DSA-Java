<h1>🚀 DSA-Java</h1>

<p>
A structured collection of <b>Data Structures & Algorithms problems solved in Java</b>, 
organized by topic for easy practice and revision.
</p>

<hr>

<h2>📚 Topics Covered</h2>

<h3>🔢 Arrays</h3>
<ul>
  <li>Check if one array is a rotation of another</li>
  <li>Find second largest element without sorting</li>
  <li>Two Sum problem</li>
</ul>

<h3>🔤 Strings</h3>
<ul>
  <li>First unique character</li>
  <li>Check if two strings are anagrams</li>
  <li>Palindrome check</li>
  <li>Longest substring without repeating characters</li>
  <li>Most frequent character</li>
  <li>Remove duplicate characters</li>
  <li>Reverse a string</li>
</ul>
